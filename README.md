# Audio Eq Visualizer

Audio Eq Visualizer, with Phase Vocoder Processing in Rust WASM.

> Note: Before running, check on how to build Rust code [here.](/wasm-fft/README.md)

To install dependencies:

```bash
npm i
```

To run:

```bash
npm run watch
```
